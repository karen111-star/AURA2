# 🧘 AURA Spa - Sitio Web Premium de Bienestar

Sitio web profesional para AURA Spa, enfocado en bienestar, relajación y experiencias sensoriales personalizadas.

## ✨ Características

- **Logo PNG integrado** desde el repositorio
- **Hero section mejorado** con estadísticas centradas
- **Menú móvil funcional** totalmente responsivo
- **Animaciones smooth** al scroll (reveal effects)
- **SEO optimizado** con meta tags Open Graph
- **Performance optimizado** con lazy loading
- **Accesible** con ARIA labels y navegación por teclado
- **Responsive** para móvil, tablet y desktop

## 📂 Estructura del Proyecto

```
AURA2/
├── index.html           # HTML principal
├── style.css            # Estilos globales
├── logo.png             # Logo oficial
├── js/
│   └── main.js         # JavaScript funcional (menú, animaciones, etc)
├── css/
│   └── variables.css   # Variables de diseño (colores, tipografía, etc)
├── tipografia/
│   └── header.css      # Estilos del header y menú
├── movil/
│   └── hero.css        # Estilos del hero section
├── animada/
│   └── sections.css    # Estilos de secciones y animations reveal
└── README.md           # Este archivo
```

- **Serif:** Cormorant Garamond (headings)
